Traduction Française - Monster Hunter RimWorld
================================================

Cette traduction complète couvre l'intégralité du contenu du mod Monster Hunter RimWorld.

Contenu traduit :
-----------------
✓ 53 monstres (petits monstres, wyvernes, dragons anciens)
✓ 40+ cuirs de monstres
✓ 47 œufs de monstres
✓ 40+ tapis de monstres
✓ Bâtiments de production (forge de chasseur, BBQ, etc.)
✓ Recherches et technologies
✓ Recettes de cuisine
✓ Projectiles et types de dégâts
✓ Événements et quêtes
✓ Intégration Pawnmorpher (mutations, morphes, sérums)
✓ Pensées et souvenirs
✓ Définitions de travail

Structure :
-----------
- Keyed/ : Messages de jeu dynamiques (2 fichiers)
- DefInjected/ : Traductions de définitions (52 fichiers, 19 types de Defs)

Total : 53 fichiers XML de traduction

Version : Compatible avec RimWorld 1.3, 1.4, 1.5, 1.6

Note : Les noms de monstres sont conservés en anglais (Rathalos, Jagras, Tigrex, etc.)
car ce sont des noms propres issus de la franchise Monster Hunter.
